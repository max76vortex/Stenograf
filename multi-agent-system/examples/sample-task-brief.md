# Sample Task Brief

## Задача

Добавить в проект служебную страницу или команду, которая показывает:

- текущий статус системы;
- наличие открытых блокирующих вопросов;
- список последних созданных артефактов активного прогона.

## Цель dry run

Эта задача достаточно маленькая, чтобы:

- пройти полный контур аналитики, архитектуры и планирования;
- не требовать большого объема кода;
- проверить, умеет ли система держать артефакты и подтверждения по этапам.

## Ограничения

- нельзя ломать существующую инфраструктуру проекта;
- нельзя вводить тяжелую новую инфраструктуру без необходимости;
- нужно использовать уже созданный контур `multi-agent-system/`.
