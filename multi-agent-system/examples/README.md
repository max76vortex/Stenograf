# Examples

Примеры нужны для первого dry run системы, когда оператор хочет проверить
оркестрацию на небольшой, но реалистичной задаче.

Рекомендуемый порядок:

1. скопировать `sample-task-brief.md` в `../current-run/task_brief.md` и отредактировать под задачу;
2. передать его оркестратору как постановку;
3. пройти этапы до планирования;
4. при необходимости продолжить до разработки.
