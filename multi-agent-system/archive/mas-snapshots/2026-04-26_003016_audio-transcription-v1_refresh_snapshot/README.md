# Current Run

Эта папка хранит артефакты активной задачи.

При старте новой задачи:

1. Очистите или заархивируйте результаты прошлого прогона.
2. Обновите `technical_specification.md`, `architecture.md`, `plan.md` и другие
   файлы под новую постановку.
3. Используйте `tasks/`, `reviews/` и `reports/` для промежуточных результатов.

## Правила

- Один активный прогон на одну задачу.
- Все блокирующие вопросы фиксируются в `open_questions.md`.
- Все task-файлы хранятся в `tasks/`.
- Все отчеты ревью и тестирования складываются в `reviews/` и `reports/`.
