# MAS snapshots (`mas-snapshots`)

Сюда скрипт **`multi-agent-system/tools/mas-new-run.ps1`** копирует полную копию папки **`current-run/`** перед «чистым стартом», если включён режим резервной копии (по умолчанию включён).

Имя каталога: `YYYY-MM-dd_HHmmss_<mas-project-id>_previous/`

Удалять вручную не обязательно; при нехватке места можно архивировать или удалять старые снимки.
