# Templates

Шаблоны для ключевых артефактов активной задачи.

Использование:

1. При старте новой задачи копируйте или адаптируйте нужный шаблон в
   `multi-agent-system/current-run/`.
2. Агент должен придерживаться структуры шаблона, но может добавлять секции,
   если это помогает качеству результата.
