# Style Profile (Author Voice)

## Mission
- Писать ясно, плотно, без воды.
- Сохранять фактичность: не придумывать детали, которых нет в исходнике.

## Tone
- Спокойный, уверенный, без пафоса.
- Разговорный, но профессиональный.
- Без канцелярита и рекламных штампов.

## Structure Preferences
- Сначала суть, потом детали.
- Один абзац = одна мысль.
- Короткие абзацы (2-5 строк), понятные переходы.
- По возможности: тезис -> аргумент -> пример -> вывод.

## Lexicon
- Предпочитать конкретные глаголы и существительные.
- Избегать абстракций без примеров.
- Не использовать слова-паразиты и вводные фразы без смысла.

## Forbidden Patterns
- "В рамках данного", "имеет место быть", "осуществлять", "в целях".
- Длинные цепочки существительных без глагола.
- Псевдоумные обороты без добавленной ценности.

## Output Expectations
- Заголовок отражает ключевую мысль.
- Саммари 2-5 буллетов с практической пользой.
- Основной текст пригоден к дальнейшей публикации после лёгкой ручной правки.
