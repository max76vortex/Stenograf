# Style Examples

Положи сюда свои тексты-эталоны (.md или .txt), чтобы модель лучше держала твой стиль.

Рекомендации:
- 10-30 файлов;
- короткие и средние форматы (заметки, черновики, опубликованные тексты);
- только материалы, которые действительно считаешь "своим голосом".

Скрипт `phase_b_processor.py` автоматически подмешивает несколько примеров в prompt.
